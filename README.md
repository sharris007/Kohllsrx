# Survey Maker

A SurveyJS Designer–style survey builder built with React, Next.js, Styled Components, and Node.js. The UI replicates the [SurveyJS Create Free Survey](https://surveyjs.io/create-free-survey) interface with matching styles and themes.

## Features

- **Top navigation bar**: SurveyJS branding, Products/Demos/Developers menus, Cart, Log In, View Demo
- **Creator tabs**: Designer, Preview, Themes, Logic, JSON Editor, Translations, Embed Survey
- **Left toolbox**: Searchable list of question types (Radio, Checkboxes, Dropdown, Slider, etc.)
- **Central canvas**: Empty state with “Add Question” and drag-and-drop support
- **Right properties panel**: Survey title, description, language, width mode, checkboxes

## Tech Stack

- **React** 18
- **Next.js** 14 (App Router)
- **Styled Components**
- **TypeScript**
- **Node.js** (Next.js API routes)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` – Start development server
- `npm run build` – Production build
- `npm run start` – Start production server
